Neste projeto, nos propomos aplicar as bibliotecas de processamento natural de linguagem, para analisar a similaridade entre vídeos de YouTube pode ser analisada unicamente através das lenas dos vídeos.

Para esta finalidade passamos pelas seguintes etapas:

- Raspagem de dados, coletando as 22 lendas de vídeos de dois canais de divulgação de  ciência populares no YouTube, o canal ciência todo dia, do  qual coletamos 11 vídeos e o canal Nerdologia do, qual coletamos 11 vídeos.

- Preprocessamento dos textos, eliminando símbolos, tokenizando, lematizando.

- Construção de uma matriz de texto por palavras, de 22 linhas e 2205 colunas de correspondendo a quantidade de palavras diferentes presentes no texto após de limpeza.

- Análise de componentes principais (PCA) nas, explicando detalhadamente o problema de encontrar as direções de maior variabilidade, a justificativa matemática para a resolução de este problema e  a aplicação no Python.

- Finalmente observamos graficamente que duas componentes principais conseguem separar vem os vídeos dos dois canais, mesmo que explicando somente o 22% da variança.
